# Projeto Engajamento e Permanência Estudantil

## 📌 Sobre o projeto
Este projeto tem como objetivo analisar dados acadêmicos e financeiros para identificar a probabilidade de evasão ou permanência de alunos.

## 🎯 Objetivo
Criar um score de engajamento estudantil baseado em dados.

## 🏫 Organização
